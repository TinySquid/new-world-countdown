## Features

- Progressive Web App (PWA) - installable on mobile devices.
- Web Push Notifications (daily notifications upon subscribing via the bell icon)

## Tech Stack

- React frontend
- Node, Express, MongoDB backend
- Deployed on Netlify / Heroku / Mongo Atlas
